Aqui está um exemplo de prompt (em português) que você pode usar no Replit para instruir uma IA ou um desenvolvedor a redesenhar o layout do seu site, garantindo que todo o contexto orne perfeitamente com o logotipo:

```
Você é um assistente de IA especialista em design de interfaces web inovadoras. Seu objetivo é redesenhar o layout do site hospedado no Replit, garantindo que cores, tipografia e estrutura reflitam com fidelidade o logotipo da marca.

Contexto
- O site está em Replit (HTML, CSS, JS).
- O logotipo está em /public/logo.png e usa as cores primárias #1A73E8 (azul) e #F9A825 (amarelo).

Objetivos
1. Harmonizar todo o visual com o logo.
2. Criar uma experiência moderna, limpa e responsiva.
3. Entregar código limpo, semântico e acessível.

Tarefas
1. Definir variáveis CSS com a paleta do logo:
   --primary-color: #1A73E8;
   --secondary-color: #F9A825;
   --neutral-light: #F5F5F5;
   --neutral-dark: #333333;
2. Tipografia:
   - Fonte principal: “Inter”, sans-serif.
   - Definir escala tipográfica para h1–h6, parágrafos e botões.
3. Layout e componentes:
   - Header fixo, fundo semi-transparente com --primary-color e logo à esquerda.
   - Navbar responsiva (mobile → menu “hamburger”).
   - Seção hero usando cor complementar ou versão SVG do logo como background.
   - Grid de cards para produtos/serviços, espaçamento consistente.
   - Footer escuro (--neutral-dark) com links, ícones sociais e créditos.
4. Responsividade:
   - Mobile: <600px.
   - Tablet: 600–1024px.
   - Desktop: >1024px.
5. Acessibilidade:
   - Contraste mínimo 4.5:1.
   - Uso de tags semânticas e atributos ARIA onde necessário.

Entregáveis
- index.html e styles.css atualizados.
- Trechos de código HTML/CSS destacados.
- Breve instruções de deploy no Replit (paths, imports e settings).

Instruções adicionais
- Comente o código explicando a vinculação de cada cor ao logo.
- Mantenha a estrutura de pastas organizada: assets/, styles/, components/.
- Forneça sugestões de animações suaves (transições de hover, fade-in).
```

Basta colar esse prompt na sua instância de ChatGPT (ou outra IA suportada no Replit) e executar. Ele guiará a ferramenta a refatorar o layout de forma consistente com o seu logotipo.
